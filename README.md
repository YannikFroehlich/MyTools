# 🧰 MyTools

**MyTools** ist ein persönliches Django-Dashboard mit kleinen, nützlichen Tools für den Alltag.  
Die App kombiniert eine moderne Startseite, eigene Schnellverknüpfungen, Wetterdaten, Dark Mode und eine übersichtliche Oberfläche in einem Projekt.

> Ein kleines Tool-Center, das nach und nach wachsen kann.

---

## ✨ Features

### 🔍 Google-Suche

Direkt auf der Startseite befindet sich eine eigene Suchleiste, mit der man schnell über Google suchen kann.

- automatische Fokussierung beim Öffnen der Startseite
- eigene Suchvorschläge
- modernes Design
- Dark-Mode-Unterstützung

---

### 🔗 Eigene Verknüpfungen

Auf der Startseite können eigene Schnellzugriffe gespeichert werden.

Beispiele:

- YouTube
- Twitch
- GitHub
- CasaOS
- Pi-hole
- Django Docs
- eigene Webseiten

Die Verknüpfungen werden im Django-Backend gespeichert und bleiben dauerhaft erhalten.

---

### 🗂️ Eigene Bereiche

Verknüpfungen können in eigene Bereiche sortiert werden.
